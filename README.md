# 🌙 Amanah — Islamic Personal Finance Tracker

> "Allah has permitted trade and forbidden riba." — Al-Baqarah: 275

A halal-first, privacy-first personal finance app for Muslim professionals. Built with React Native + Expo.

## Features

- **Dashboard** — Net worth, monthly summary, financial freedom progress, Hijri date
- **Transactions** — Track income & expenses with halal-only categories
- **Zakat Calculator** — Assets, liabilities, nisab check, 2.5% calculation, payment history
- **Halal Portfolio** — Gold, Sukuk, Syariah mutual funds, property — no riba assets
- **Financial Freedom Planner** — Target amount, halal return rate, years-to-goal projection

## Stack

React Native + Expo · TypeScript · Zustand · AsyncStorage

## Philosophy

- 🚫 No riba — only halal asset types
- 🔒 Privacy-first — all data on device, no backend
- 💰 One-time purchase — no subscriptions, no ads

## Getting Started

```bash
git clone https://github.com/achmadnaufal/amanah
cd amanah && npm install && npx expo start
```
